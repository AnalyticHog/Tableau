This repository contains a collection of Tableau data visualisation projects.
Each project was created to practise and demonstrate skills in interactive dashboards, data exploration, and storytelling with data.

## Contents
- Interactive dashboards

- Data exploration and insights

- Map‑based visualisations

- Trend and comparison analysis

- Practice datasets

## Tools Used
- Tableau Desktop / Tableau Public

- Excel / CSV datasets

- Basic data cleaning and preparation
